# Visualizer

This folder holds interactive simulations used throughout the curriculum
(e.g. the swerve drive visualizer for the Swerve & Odometry training section).

Each simulation should be self-contained: its own subfolder with an
`index.tsx`/`index.jsx` and a `styles.module.css`, following the pattern
used by `JavaRunner` and `ControlMethodChooser` elsewhere in `components/`.
